# Why I Started Ngeseries

After finishing my second semester, I decided to build a habit:
write consistently and learn in public.

This website is the result of that decision. I call it **Ngeries** a small writing series where I document what I learn while studying and building things.

It has three tracks:

## 1. Ngemath
Focused on mathematics, especially:
- linear algebra
- vectors
- mathematical foundations for Machine Learning

## 2. NgeCP
Focused on competitive programming:
- algorithms
- problem solving
- coding patterns

## 3. Ngoding
Focused on software engineering:
- backend development
- side projects
- engineering lessons

## Why writing?

Because writing forces clarity.

-> If I can explain something clearly, it usually means I understand it.

-> If I cannot, it means I still need to learn.

## My goal

Not perfection, train my consistency. One post every three days.

Let's see where this goes.
